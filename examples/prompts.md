# Prompt Examples

The following prompts can be directly copied into Codex/Antigravity for use.
## Only do picture planning

```text
Use $zaizai-bean-illustrations. Do not generate raw images yet.
Please analyze which parts of the case study below deserve to be accompanied by images, and output a list of about 5 images.
Each image should be clearly labeled:
- After which paragraph?
- The theme of the image
- Core meaning
- Structure type
What is ZaiZai Bean doing in the picture?
- Suggested elements
- Suggested English annotation words

<Paste case study>
```

## Image accompanying the case study

```text
Use $zaizai-bean-illustrations to generate 4 whimsical illustrations for the case study below.
Requirements: 16:9 landscape, pure white background, black hand-drawn line art, and a small amount of handwritten English annotations in red, orange, blue.
Each image should only illustrate one core structure; avoid using PowerPoint infographics or cute cartoons.

<Paste case study>
```

## Long case study with pictures strategy

```text
Use $zaizai-bean-illustrations to create illustrations for the long case study.
Avoid using images evenly; instead, focus on key cognitive anchors: key decisions, product tradeoff, user pain points, product strategy shifts, user journeys, feedback loops, system bottlenecks, input/output systems.
The default is 6-8 images. Output a sshot list first, without generating images.

<Paste case study>
```
